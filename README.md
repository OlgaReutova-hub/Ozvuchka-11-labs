Этот чат бот создан для озвучки текстов text2speech при помощи Elevenlabs
Он представляет собой Python-приложение — Telegram-бот, который озвучивает введённый пользователем текст с помощью технологии синтеза речи ElevenLabs API.
Бот позволяет выбрать тип голоса (мужской или женский), прослушать примеры голосов и получить готовое аудио или голосовое сообщение прямо в Telegram.

**Возможности**

Выбор типа голоса — бот предлагает выбрать мужской или женский голос.
Прослушивание примеров — перед выбором можно послушать, как звучит каждый голос.
Озвучка текста — бот генерирует аудиофайл (MP3) и отправляет его пользователю.
Поддержка нескольких моделей ElevenLabs — можно использовать разные языки и стили речи.
Отправка голосового сообщения — результат синтеза дублируется в формате voice-сообщения.

**Используемые технологии**

Python — основной язык разработки.
PyTelegramBotAPI — для интеграции с Telegram Bot API.
ElevenLabs Python SDK — для синтеза речи через ElevenLabs API.
Virtualenv — для изоляции зависимостей.

**Принцип работы**

Пользователь отправляет команду /start.
Бот предлагает выбрать тип голоса (мужской / женский).
Пользователь прослушивает примеры доступных голосов.
После выбора бот запоминает голос и предлагает ввести текст.
ElevenLabs API генерирует аудио, которое бот отправляет как MP3 и как голосовое сообщение.

**Необходимые разрешения ElevenLabs API**

Для корректной работы необходимо включить в настройках API-ключа следующие права:
Text to Speech → Access
Voices → Read
Voice Generation → Access
